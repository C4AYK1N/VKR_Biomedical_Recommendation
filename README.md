# Рекомендательная система для анализа биомедицинских данных

## Описание проекта
Система для классификации сердечных аритмий по данным ЭКГ с использованием машинного обучения.

## Функционал
- Загрузка и предобработка данных ЭКГ
- Обучение моделей Random Forest и CNN
- Прогнозирование типов аритмий
- Генерация медицинских рекомендаций
- Визуализация результатов

## Установка
```bash
git clone <репозиторий>
cd VKR_Biomedical_Recommendation
pip install -r requirements.txt
streamlit run main.py
